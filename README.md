# 坦克大战 HTML5 游戏

一个基于HTML5 Canvas开发的经典坦克大战游戏。

## 游戏特点

- 玩家控制绿色坦克与红色敌人坦克战斗
- 多关卡系统，难度递增
- 砖块可破坏，边墙不可破坏
- 计分系统和生命值系统
- 现代化的UI设计

## 如何运行

1. 下载 `index.html` 文件
2. 用现代浏览器打开
3. 开始游戏！

## 游戏控制

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| A / ← | 向左移动 |
| S / ↓ | 向下移动 |
| D / → | 向右移动 |
| 空格 | 射击 |
| R | 重新开始 |

## 游戏规则

- 消灭所有敌人坦克进入下一关
- 每消灭一个敌人获得 100 分
- 完成关卡获得 500 分奖励
- 玩家有 3 条生命
- 被敌人子弹击中会损失生命

## 技术栈

- HTML5 Canvas
- 原生 JavaScript
- CSS3
